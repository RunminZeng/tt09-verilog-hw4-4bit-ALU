## How it works

The design 4-bit ALU includes the function of:
1. addition
2. subtraction
3. multiplication
4. division
5. logical AND
6. logical OR
7. logical XOR
8. logical NOT
9. Encryption operation
It operation is controlled by the input of OPCODE.
